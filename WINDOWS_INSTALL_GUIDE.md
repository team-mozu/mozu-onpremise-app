# 🏫 Mozu 모의주식 실습환경 - Windows 설치 가이드

## 📋 설치 전 준비사항

### 필수 소프트웨어
다음 프로그램들이 필요합니다 (설치 시 자동으로 확인됩니다):

1. **Java 17 이상** - 서버 실행용
   - 다운로드: https://adoptium.net
   - 권장: Eclipse Temurin 17 LTS

2. **Git** - 프로젝트 다운로드용
   - 다운로드: https://git-scm.com/download/win

3. **MySQL** - 데이터베이스 (옵션)
   - 다운로드: https://dev.mysql.com/downloads/mysql/

## 🚀 설치 과정

### 1단계: 설치 파일 다운로드
- `mozu-onpremise-setup-0.1.0.exe` 파일을 다운로드합니다

### 2단계: 설치 실행
1. 다운로드한 `.exe` 파일을 **우클릭** → **관리자 권한으로 실행**
2. Windows에서 "알 수 없는 게시자" 경고가 나오면 **"추가 정보"** → **"실행"** 클릭
3. 설치 경로를 선택합니다 (기본: `C:\Program Files\Mozu 모의주식 실습환경`)
4. **"바탕화면에 바로가기 만들기"** 체크
5. 설치 완료

### 3단계: 첫 실행
1. 바탕화면의 **"Mozu 모의주식 실습환경"** 아이콘을 더블클릭
2. 필요한 소프트웨어가 없으면 다운로드 링크가 제공됩니다
3. **MySQL 비밀번호 설정** (비어두거나 설정한 비밀번호 입력)
4. **프로젝트 저장 위치 선택** (권장: `C:\mozu-projects`)
5. **"🚀 모의주식 실습환경 시작"** 버튼 클릭

## 🔧 문제 해결

### 자주 발생하는 문제들

#### 1. "Java가 설치되지 않았습니다" 오류
**해결방법:**
- https://adoptium.net 에서 Java 17 다운로드 후 설치
- 설치 후 컴퓨터 재시작

#### 2. "Git이 설치되지 않았습니다" 오류  
**해결방법:**
- https://git-scm.com/download/win 에서 Git 다운로드 후 설치
- 설치 시 모든 기본 옵션 그대로 설치

#### 3. "MySQL 연결 실패" 오류
**해결방법:**
- MySQL이 설치되지 않은 경우: https://dev.mysql.com/downloads/mysql/
- 비밀번호를 잘못 입력한 경우: 앱에서 올바른 비밀번호 입력
- MySQL 없이 사용하려면: 비밀번호 칸을 비워두고 실행

#### 4. "포트가 이미 사용 중입니다" 오류
**해결방법:**
1. 작업 관리자(Ctrl+Shift+Esc) 열기
2. "프로세스" 탭에서 `java` 또는 `node` 프로세스 찾아서 종료
3. 앱 재시작

#### 5. 브라우저에서 "사이트에 연결할 수 없음" 오류
**해결방법:**
1. 앱이 완전히 시작될 때까지 기다리기 (약 1-2분)
2. 로그에서 "✅ 서버 실행중" 확인
3. 브라우저 새로고침 (F5)
